server.c : 處理伺服器的運作
client.c : 處理用戶端的運作
執行方式
1.make編譯
2.在一個終端機執行./server
3.開另外幾個終端機./client 127.0.0.1


使用方法:

更改名字:1 要更改成什麼名字

顯示使用者:2

邀請下棋:3 我的名字  對方的名字

接受與否:
接受:5 1 邀請者名字
不接受:5 0 邀請者名字

下棋方式:
棋盤格式:

1,1 1,2 1,3
2,1 2,2 2,3
3,1 3,2 3,3

若要在該處下棋
輸入該座標即可（ex:1,1）

登出:quit
